# Knuts & Bolts

Knuts & Bolts is a library written in Typescript, that implements basic data structures and algorithms.

## Installation

Clone this repository

```git@github.com:Jessica-Koch/knuts-and-bolts.git```

## Testing
Knuts&Bolts uses Jest to run its tests.

Make sure to run the build,
`npm run build`

Now you're ready to run the tests, `npm run test`.  By default, the `--watch` flag is on, if you prefer to run the tests yourself, simply go to the `package.json` and change `"test": "jest --watch --coverage --colors",` to `"test": "jest --coverage --colors",`
